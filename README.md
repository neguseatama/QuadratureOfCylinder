## Overview
* C言語による円柱の求積(表面積と体積)

## Description
* C言語による円柱の求積(表面積と体積)のプログラムです。
* Xcodeで動作を確認しました。
* 円の半径と円柱の高さの入力を受け付けることで、それぞれの演算の結果を出力させられます。

## Note
* 円柱の表面積と体積の公式を利用しました。
* 円周率は3.14としています。
